# Python Projects

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

This repository contains a small collection of beginner-friendly Python command-line programs. Each script is self-contained and prompts for input directly in the terminal.

## Included Projects/Programs

- `calculator.py` - basic arithmetic calculator for addition, subtraction, multiplication, and division.
- `number_guessing.py` - number guessing game that gives hints until the correct answer is found.
- `palindrome_checker.py` - checks whether an entered string reads the same forwards and backwards.
- `word_counter.py` - counts the number of words in an entered sentence.

## Requirements

- Python 3.13

## How to Run

Open a terminal in the project folder and run one script at a time:

```bash
python calculator.py
python number_guessing.py
python palindrome_checker.py
python word_counter.py
```
